# Smart Pet Care System – Minbe

## Project Description

This project is a **Smart Pet Care System website** designed to help pet owners monitor their pet’s health and access pet-related services in one place.

The system uses a **wearable collar device** for pets. The collar contains sensors that measure important health parameters. These sensor readings are sent to the website through an **API**, allowing pet owners to monitor their pet's condition online.

## Key Features

* Wearable **smart collar** for pets
* Sensors to monitor pet health parameters
* Data transmission to the website through API
* Health monitoring dashboard for pet owners
* Online platform for **pet accessories and services**

## Technologies Used

* HTML
* CSS
* JavaScript
* API integration

## How the System Works

1. The pet wears a **sensor-enabled smart collar**.
2. Sensors collect health data from the pet.
3. The data is sent to the server using an **API**.
4. The website displays the pet's health information for the owner.
5. Owners can also browse and purchase **pet-related accessories** on the platform.
